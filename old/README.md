# UO Commander

A modern macOS application for administering ModernUO Ultima Online servers, built with SwiftUI and a custom HTTP API module.

## Features

### 🔐 Authentication & Authorization
- Secure JWT-based authentication
- Username/password login with GameMaster+ access requirement
- Secure credential storage in macOS Keychain
- Touch ID support (future)

### 📊 Dashboard
- Real-time server status (players, uptime, memory, CPU)
- Quick action buttons for common operations
- Online player overview
- Server metrics and statistics

### 👥 Player Management
- **Search & Filter**: Find players by name, account, or IP
- **Inspect Players**:
  - Character overview (stats, location, playtime)
  - Equipment viewer (visual grid with item details)
  - Skills list (sortable by value)
  - Properties inspector
  - Speech history
  - Hardware/client info
- **Actions**:
  - Kick players (with reason)
  - Ban/unban accounts
  - Jail/unjail players
  - Squelch/unsquelch
  - View account details

### 📋 Account Management
- Search accounts by username or IP
- View account details (characters, login history, comments)
- Ban/unban accounts
- Change access levels
- Add/edit comments

### 🖥️ Server Control
- **Broadcast Messages**:
  - Send to all players or staff only
  - Message preview
- **World Save**: Manual save trigger
- **Shutdown/Restart**:
  - Visual countdown timer
  - Configurable delay (30s, 1m, 2m, 5m, custom)
  - Save before action toggle
  - Cancel during countdown
- ~~**Server Lockdown**~~: DISABLED - AccountHandler.LockdownLevel not available in ModernUO

### ~~🔥 Firewall Management~~ - DISABLED
Firewall endpoints are disabled because Firewall type doesn't exist in ModernUO
- ~~View all firewall rules~~

- ~~Quick block/unblock player IP~~
- ~~Check if IP is blocked~~

### 📝 Logs Viewer
- Real-time server log streaming
- Filter by log level (Info, Warning, Error)
- Search functionality
- Export logs

## Architecture

### Two-Part System

```
┌─────────────────────────┐         ┌─────────────────────────┐
│   ModernUO Server       │         │   UO Commander (macOS)  │
│                         │         │                         │
│  ┌───────────────────┐  │  HTTP   │  ┌───────────────────┐  │
│  │  HTTP API Module  │◄─┼─────────┼─►│  SwiftUI App      │  │
│  │  (C# / .NET 8)    │  │  JSON   │  │  (Swift 5.9+)     │  │
│  └───────────────────┘  │         │  └───────────────────┘  │
│                         │         │                         │
│  - JWT Authentication   │         │  - Authentication       │
│  - Player Management    │         │  - Player Management    │
│  - Server Control       │         │  - Server Control       │
│  - Firewall             │         │  - Dashboard            │
└─────────────────────────┘         └─────────────────────────┘
```

### Server-Side (ModernUO HTTP API)

**Files:**
- `HttpApiServer.cs` - Main HTTP server implementation
- `JwtHelper.cs` - JWT token generation and validation

**Configuration** (in `modernuo.json`):
```json
{
  "httpApi": {
    "enabled": true,
    "port": 8081,
    "jwtSecret": "<auto-generated>",
    "jwtExpiryHours": 24
  }
}
```

**API Endpoints:**

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/auth/login` | POST | Login with credentials |
| `/api/auth/verify` | GET | Verify JWT token |
| `/api/auth/logout` | POST | Logout and invalidate token |
| `/api/server/status` | GET | Get server status |
| `/api/server/save` | POST | Trigger world save |
| `/api/server/shutdown` | POST | Shutdown server |
| `/api/server/restart` | POST | Restart with countdown |
| `/api/server/broadcast` | POST | Broadcast message to all |
| `/api/server/staff-message` | POST | Message to staff only |
| `/api/players` | GET | List online players |
| `/api/players/search` | GET | Search players by name |
| `/api/players/{serial}/kick` | POST | Kick player |
| `/api/players/{serial}/ban` | POST | Ban player account |
| `/api/players/{serial}/unban` | POST | Unban player account |
| `/api/players/{serial}/equipment` | GET | Get equipped items |
| `/api/players/{serial}/backpack` | GET | Get backpack contents |
| `/api/players/{serial}/skills` | GET | Get player skills |
| `/api/players/{serial}/properties` | GET | Get player properties |
| `/api/accounts/search` | GET | Search accounts |
| `/api/accounts/{username}/ban` | POST | Ban account |
| `/api/accounts/{username}/unban` | POST | Unban account |
| ~~`/api/firewall`~~ | ~~GET~~ | ~~List firewall rules~~ - DISABLED |

## Installation

### Part 1: Install ModernUO HTTP API Module

1. **Copy API Module Files**
   ```bash
   cp HttpApiServer.cs /path/to/ModernUO/Projects/Server/HTTP/
   cp JwtHelper.cs /path/to/ModernUO/Projects/Server/HTTP/
   ```

2. **Register the Module**
   
   In `Projects/Server/Main.cs`, add to the `Configure` method:
   ```csharp
   HttpApiServer.Configure();
   ```
   
   And in `EventSink_ServerStarted`:
   ```csharp
   _ = HttpApiServer.Start();
   ```

3. **Enable in Configuration**

   Add to `modernuo.json`:
   ```json
   {
     "httpApi": {
       "enabled": true,
       "port": 8081
     }
   }
   ```

4. **Build and Run ModernUO**
   ```bash
   dotnet build
   ./ModernUO
   ```

### Part 2: Build UO Commander macOS App

**Requirements:**
- macOS 14.0 (Sonoma) or later
- Xcode 15.0 or later
- Swift 5.9+

**Build:**
```bash
cd uo-commander
swift build
```

**Run:**
```bash
swift run UOCommander
```

**Or open in Xcode:**
```bash
swift package generate-xcodeproj
open UOCommander.xcodeproj
```

## Usage

1. **Launch UO Commander**
2. **Enter Server URL**: `http://localhost:8081` (or your server's IP)
3. **Login**: Use an account with GameMaster+ access level
4. **Start Administering**: Use the sidebar to navigate between features

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘ + S` | Save world |
| `⌘ + B` | Broadcast message |
| `⌘ + R` | Restart server |
| `⌘ + F` | Search players |
| `⌘ + 1-6` | Switch tabs |

## Security

- **JWT Authentication**: All API endpoints require valid JWT token
- **Access Control**: Enforces ModernUO access levels (GameMaster+)
- **Keychain Storage**: Credentials stored securely in macOS Keychain
- **HTTPS Support**: Recommended for production use
- **Rate Limiting**: Built-in protection against brute force attacks
- **Audit Logging**: All admin actions logged with timestamp and actor

## Configuration

### Server-Side (modernuo.json)

```json
{
  "httpApi": {
    "enabled": true,
    "port": 8081,
    "requireHttps": false,
    "jwtExpiryHours": 24,
    "jwtSecret": "<random-string>",
    "allowedIPs": ["127.0.0.1", "192.168.1.0/24"],
    "rateLimit": {
      "loginAttempts": 5,
      "loginWindowMinutes": 15
    }
  }
}
```

### Client-Side

Server URL and preferences are stored in `UserDefaults` and can be configured in the app settings.

## API Documentation

### Authentication

**Login:**
```bash
curl -X POST http://localhost:8081/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"Admin","password":"secret"}'
```

**Response:**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "username": "Admin",
  "accessLevel": 4,
  "expiresHours": 24
}
```

### Get Server Status

```bash
curl http://localhost:8081/api/server/status \
  -H "Authorization: Bearer <token>"
```

**Response:**
```json
{
  "isRunning": true,
  "uptime": 3600.5,
  "playerCount": 15,
  "maxPlayers": 4096,
  "memoryUsage": 524288000,
  "worldSaveStatus": "Idle",
  "version": "1.0.0",
  "lockdownLevel": "None"
}
```

### Broadcast Message

```bash
curl -X POST http://localhost:8081/api/server/broadcast \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"message":"Server will restart in 5 minutes"}'
```

### Ban Player

```bash
curl -X POST http://localhost:8081/api/players/0x12345/ban \
  -H "Authorization: Bearer <token>"
```

## Troubleshooting

### Common Issues

**"Invalid credentials"**
- Check username/password spelling
- Verify account has GameMaster+ access level
- Check ModernUO server logs for authentication errors

**"Connection refused"**
- Verify HTTP API is enabled in `modernuo.json`
- Check port number is correct
- Ensure ModernUO server is running

**"Insufficient privileges"**
- Account must be GameMaster (2) or higher
- Check account's AccessLevel in ModernUO

### Logs

Server-side logs appear in ModernUO console with `[HTTP API]` prefix.
